# pyqzan - 朋友圈集赞截图生成器

微信朋友圈集赞截图生成工具，一键生成点赞评论截图，支持多种样式。

## 在线使用

### GitHub Pages（仅截图功能）
https://dengxiaoyuovo.github.io/pyqzan/

### Vercel（全功能：截图 + 获取公众号信息）

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/git/external?repository-url=https://github.com/DengxiaoyuOVO/pyqzan)

点击上方按钮，用 GitHub 账号登录 Vercel，一键部署即可获得专属域名。部署后：
- 截图生成功能完整可用
- 「获取公众号信息」功能也可用（Vercel 提供后端支持）
- 零维护，自动跟随仓库更新

## 本地运行

```bash
npm install
npm run serve
```

访问 http://localhost:3456

后端：
```bash
cd weinode && npm install && node main.js
```
